---
title: Google BigQuery connector
description: Learn how to read Delta Lake tables from Google BigQuery.
---

Google BigQuery supports reading Delta Lake (reader version 3 with [Deletion Vectors](/apache-spark-connector/what-are-deletion-vectors) and [Column Mapping](/apache-spark-connector/delta-column-mapping/)). Please refer to [Delta Lake BigLake tables documentation](https://cloud.google.com/bigquery/docs/create-delta-lake-table) for more details.
